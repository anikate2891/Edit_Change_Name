# Edit_Change_Name

Simple front-end demo to edit and preview name fields — built with plain HTML, CSS and JavaScript.

## What it does
- Provides a minimal UI to edit a name and see changes live.
- Lightweight and easy to integrate into small projects or use as a learning example for DOM manipulation.

## Tech
- HTML
- CSS
- JavaScript (vanilla)

## Files
- `index.html` — main UI
- `style.css` — styles
- `script.js` — behavior
